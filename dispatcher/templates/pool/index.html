About

The server for generating search options for OnionHarvester's project explained here.
Clients that connect to this server will get start and end onion address, scanning ports and timeout.
Then they will check all addresses in the provided range for specific ports on TOR network and will return search results in a json format including valid onion addresses, their ports and time of scan.
The server can show online searches and top 15 results got from clients that you can see them on the Results tab.
